Cognis-lab 規約（v1.3）
制定日：2026年4月17日 主宰者：枝成 豆太 公開：GitHub / note

第1条（名称・所在地）
1.	本団体は「Cognis-lab（コグニスラボ）」と称する。
2.	略称は「CL」とする。
3.	本団体は物理的施設を有さない。
4.	本団体の主たる事務所は東京都内とし、詳細所在地は主宰者が管理する。
5.	対外的活動において「コグニス＠cognislab」の呼称を使用できる。

第2条（目的）
本団体は、情報過多社会における個人の判断能力・精神的安定・創造性を保護するため、以下の活動を行う：
1.	内部安全工学（ISE：Inner Safety Engineering）の研究開発
2.	認知OS「MiLAS/AOS」の設計・実装・普及
3.	認知健全性指標CIE（Cognitive Integrity Estimate）の研究開発
4.	ソフトウェア研究開発および情報発信
5.	本団体は特定の思想・状態・価値観への到達を目的としない。

第3条（主宰者・構成員）
1.	主宰者は枝成 豆太とする。
2.	主宰者は団体運営の最終意思決定権および知的財産の管理権を有する。
3.	構成員は主宰者の承認により参加する。
4.	構成員は各自のISEを尊重し、互いの判断能力および精神的安定を毀損しない。
5.	構成員は対外的活動において主宰者の事前承認なく団体名義を使用しない。
6.	個人の発言は団体の公式見解を代表しない。
7.	構成員は本団体の財産に対して持分を有しない。
8.	構成員は本団体の負債について個別の責任を負わない。
9.	ISE侵害とは、主宰者または構成員の判断能力・精神的安定を毀損する行為を指す。

第4条（意思決定）
1.	団体の最終意思決定は主宰者が行う。
2.	日常運営は主宰者と構成員の合意により行う。
3.	主宰者は構成員のISEを侵害しない範囲で意思決定を行う。

第5条（知的財産権）
1.	団体活動で生じた成果物（コード・文書・意匠等）の著作財産権は主宰者に帰属する。
2.	構成員は著作者人格権を保持する。
3.	構成員が団体活動として創作した著作物の著作財産権は、主宰者に無償で譲渡されたものとみなす。
4.	構成員の脱退・除名後も、在籍中の成果物の権利帰属は変わらない。
5.	非営利目的での利用・改変・再配布は、出典明記を条件として自動的に許可する。
6.	商用利用は主宰者の許可を要する。
7.	MiLASマークおよびAOSバッジは本団体の公式象徴とし、無断使用を禁ずる。

第6条（資金・会計）
1.	本団体は非営利を原則とする。
2.	活動資金は主宰者および構成員の任意拠出により賄う。
3.	必要に応じて、主宰者名義の屋号付き個人口座を用いて資金管理を行う。
4.	会計は簡易簿記により主宰者が管理し、年1回公開する。

第7条（外部関係・独立性）
1.	本団体は外部の資本・政治・宗教・企業から独立する。
2.	外部提携は主宰者のISEを最優先して判断する。
3.	主宰者はISEを損なうおそれがある場合、いかなる提携・依頼も拒否できる。
4.	主宰者のISEが維持困難な場合、団体活動は自動的に縮小または停止される。
5.	活動の停止・再開は主宰者が決定する。

第8条（加入・脱退）
1.	構成員の加入は主宰者の承認による。
2.	構成員はいつでも脱退できる。
3.	ISE侵害または規約違反が認められた場合、主宰者は除名できる。

第9条（規約の改定・解散）
1.	本規約の改定は主宰者が行う。
2.	改定内容はGitHubおよびnoteで公開する。
3.	本団体の解散は主宰者の判断により行う。
4.	解散後も、本団体の知的財産権は主宰者に帰属する。


第10条（準拠法・管轄）
1.	本規約は日本法に準拠する。
2.	紛争は誠実に協議し、解決しない場合は東京地方裁判所を第一審の専属的合意管轄裁判所とする。


付則
本規約はGitHub上の公開履歴（コミットハッシュ）をもって制定記録とする。
