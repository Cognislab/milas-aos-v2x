# MiLAS AOS v2.x — Mind-Layered Architecture System
Cognis-lab / Mameta Edane  
2026

MiLAS (Mind-Layered Architecture System) is a cognitive architecture that
constructs the internal world of a person as an operating system,
and provides engineering guarantees for cognitive stability, safety,
and rebootability.

---

## 📘 MiLAS Paper Map (Overview and Roles of the Three Papers)
April 22, 2026  
Preprint published on Jxiv

### 1. MiLAS Core Model (L1/L2)
Foundational design of the cognitive structure in L1 (Stability Layer)
and L2 (Generation Layer).  
It defines the essential mechanisms for maintaining stable cognitive states
and generating cognitive processes.

### 2. MiLAS Cognitive Processing Engine (L2)
Formalization of the cognitive processing engine that operates within L2.
It describes how cognitive processes are generated, transformed,
and integrated.

### 3. MiLAS L3 Governance Layer (L3)
Formalization of the policy/governance layer that supervises and guides
cognitive processes.  
It defines how context, goals, and policies influence cognitive behavior.
